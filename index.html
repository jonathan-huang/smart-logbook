What is Logbook and what can it do?

Logbook is a clever software which functions along with gmail, it asks users for gamil addesses and then enbles logging functions, as email addresses are
saved to the server, users can log notes to the display board. The notes displayed are not permanent and will be cleared on reloading and refreshing,
but a formatted copy of note will be sent to the users' email addresses with loggeg titles and loggeg contents, as well as the time.


How to use Logbook

  • First, to enable log tracking, go to settings and find where it says Email logging and enable it by
    clicking the check box. Then, you will see the input is enabled. Type in your email address and submit,
    if it says You email has been saved, you can start the next step.
  • Then, go to logging area, the first input field is for you to enter your note title which will be replace with Untitle. if nothing is entered, the second field is for the note's contents, which will be replaced with - if nothing is entered.
  • Now choose if you want to classify your note as important or normal and... Log and have fun!
